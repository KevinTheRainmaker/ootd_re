# 시장 분석

## 시장 동향 & 타이밍

### 국내 OOTD SNS 공백 (2023~)

- **스타일쉐어 2023년 서비스 종료**: 국내 사용자 기반 OOTD 전용 SNS 부재 상태
- **공백 기간 = 2~3년** 누적: 후속 진입자 없이 공실 상태
- **MZ 트렌드**: 조용한 럭셔리, 빈티지 믹스 → 아이템 출처 정보 자체가 콘텐츠 가치
- **크리에이터 경제 성숙**: 인스타/유튜브 패션 크리에이터의 "이 옷 어디꺼?" 페인 누적

### 글로벌 트렌드

- **AI 비주얼 콘텐츠 도구 폭증**: Canva Magic, Adobe Firefly, gpt-image-1
- **소셜 카드 포맷 표준화**: 인스타 스토리/링크트리 → 1080x1920 카드가 단일 콘텐츠 단위
- **링크인바이오 시장 성장**: 크리에이터당 평균 5-10개 외부 링크 필요

### 타이밍 평가: **A (지금이 적기)**

- 국내 공백 + AI 도구 성숙 + 크리에이터 경제 = 3박자

## 경쟁사가 전부 놓치는 것

### 핵심 기회: "사진 한 장 → AI 카드 → 공유 가능한 URL"의 단일 30초 플로우

기존 경쟁사 분류:

- **A. 옷장 기록형** (Whering): 기록은 잘하지만 공유 콘텐츠 약함
- **B. SNS 카드 디자인형** (Unfold, Canva): 디자인은 좋지만 AI 분석 없고 수동 작업 10분+
- **C. 쇼핑 중심형** (에이블리, 지그재그): 정보가 풍부하지만 사용자 콘텐츠 생성 없음
- **D. 링크 도구** (링크트리): 링크만 전달, 비주얼 콘텐츠 약함

**A + B + D를 한 흐름으로 합친 도구가 없음.** 우리가 첫 진입.

### 부가 기회

1. **한국어 OOTD SNS 공백** (스타일쉐어 폐쇄 후 2년+)
2. **크리에이터 협찬 공시 도구**: "광고" 라벨을 카드에 자동 표기하면 광고 공정화법 대응 도구로도 활용 가능 (Phase 2 검토)
3. **링크인바이오 통합**: 카드 = 인스타 바이오 링크의 콘텐츠
