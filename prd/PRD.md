# PRD: OOTD-re (가칭)

> 생성일: 2026-05-28 | 버전: 1.0 | 복잡도: High
> 후속 단계: `/tth prd/ 읽고 구현해줘`

## 1. 한 줄 정의

**OOTD-re는 패션 크리에이터와 MZ세대가 사진 한 장으로 AI 손글씨 카드를 만들어 링크 하나로 착장 정보를 공유할 수 있게 해준다.**

- 입력: 착장 사진 1장
- 출력: AI가 분석한 아이템 정보 + 손글씨 스타일 주석이 그려진 카드 이미지 + 공유 가능한 URL
- 핵심 차별점: 커머스 종속 없음, 크리에이터-팔로워 간 정보 전달 도구

## 3. Six Hats 제품 평가

| Hat               | 관점          | 핵심 발견                                                                                                                                                                                                                     |

| **White (사실)**  | 시장 데이터   | 스타일쉐어 2023 종료로 국내 OOTD SNS 공백. Whering/Combyne 등 글로벌 앱은 쇼핑 종속. AI API 비용 분석 $0.003-0.01/회, 이미지 생성 $0.04-0.08/장.                                                                              |
| **Red (직감)**    | 사용자 열정   | 크리에이터의 "이 옷 어디꺼?" 댓글 페인이 강함. 인터뷰에서 1인 개발자의 도구적 직관(공유=핵심)이 명확.                                                                                                                         |
| **Black (위험)**  | 실패 시나리오 | (1) gpt-image-1 API가 원본 사진 위 주석 생성을 지원 안 함 → 핵심 가치 무너짐. (2) AI 분석 정확도가 60% 미만이면 사용자 이탈. (3) 1개월 MVP에 공유 페이지까지 무리. (4) 카드 생성 단가 폭발 시 무료 5회/유료 30회 한도가 적자. |

## 4. 제품 전략

### 포지셔닝

**"커머스에 종속되지 않은, 크리에이터를 위한 착장 콘텐츠 도구"**

- vs Whering/Combyne: 쇼핑 X, 공유 O
- vs Unfold/Canva: 패션 특화 AI 분석 O
- vs 에이블리/지그재그: 쇼핑몰 X, 개인 콘텐츠 O
- vs 스타일쉐어(폐쇄): 후계 포지션, 도구 우선 SNS 후순위

### 차별화 요소

1. **사진 → 카드 → 링크가 30초 안에 끝나는 단일 플로우** (경쟁사 평균 10-15분)
2. **AI가 사진 위에 직접 손글씨 주석을 그리는 비주얼** (Plan A 성공 시)
3. **공유 페이지 URL 자체가 콘텐츠** — 인스타 바이오, 링크트리, 스토리 링크에 박는 용도
4. **커머스 없음** — 사용자는 광고/쇼핑 없이 순수 도구로 사용

### 안티-기능 (절대 하지 않을 것 — MVP에서)

- 쇼핑몰 연동 / 제휴 링크 / 가격 비교
- 댓글, 좋아요, 팔로우 등 SNS 인터랙션
- 채팅/메시징
- 그룹/커뮤니티 기능
- AI 추천 코디 ("이 옷에 이거 매칭하세요")
- 결제 시스템 (차기 단계)
