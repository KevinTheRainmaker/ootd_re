## 용어 정의

| 용어                         | 정의                                                                                                        |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------- |
| **OOTD**                     | "Outfit Of The Day". 그날의 착장.                                                                           |
| **OOTD 레코드 (record)**     | 사용자가 저장한 하나의 착장 기록. 원본 사진 + 분석 + 카드 + 메타데이터.                                     |
| **아이템 (item)**            | OOTD 안의 개별 옷/액세서리 (상의, 하의, 신발 등). 1개 OOTD = N개 item.                                      |
| **카드 (card)**              | 원본 사진 위에 손글씨 주석이 그려진 1080x1920 PNG 이미지. SNS 공유용.                                       |
| **공유 페이지 (share page)** | `/share/[share_id]` URL. SSR + OG 메타. 비로그인 접근 가능. 크리에이터가 인스타 바이오에 박는 링크.         |
| **Plan A**                   | OpenAI gpt-image-1 API로 사진 위 손글씨 주석 직접 생성 (최대 위험, 차별화 핵심).                            |
| **Plan B**                   | html2canvas/satori + Google Fonts (Nanum Pen Script)로 클라이언트/서버 사이드 카드 합성 (fallback, 비용 0). |
| **share_id**                 | 공개 OOTD의 외부 노출 ID. nanoid 8자. 추측 불가.                                                            |
| **usage_log**                | 사용자별 월별 카드 생성 횟수 기록. 한도 제어용.                                                             |
| **free / pro**               | 사용자 플랜. free = 월 5회, pro = 월 30회. MVP는 free만 운영.                                               |
| **D7 재방문율**              | 신규 가입자 중 7일 뒤 다시 접속한 비율. 핵심 retention 지표.                                                |
| **AI 수용률**                | 분석 결과를 편집 없이 카드 생성으로 진행한 비율. 60% 목표.                                                  |
| **크리에이터 링크 박기**     | 인스타 바이오, 링크트리, 스토리 등에 우리 /share URL을 등록하는 행동. 핵심 바이럴 지표.                     |

## 리서치 출처 & 링크

### Agent A — 경쟁사 리서치

- **시장 공백 분석**: 착장기록 + AI분석 + SNS카드를 한 흐름으로 연결하는 앱 없음
- **국내**: 스타일쉐어 2023 종료, 에이블리/지그재그는 쇼핑 중심
- **글로벌**: Whering(영국, 옷장 + 쇼핑), Combyne(독일, 콜라주), Unfold(SNS 템플릿), Canva(범용)
- **링크 도구**: 링크트리, LTK — 카드 콘텐츠 없음
- **결론**: A(기록) + B(카드) + D(링크)를 한 흐름으로 합친 도구 없음

### Agent B — 기술 생태계

- **Vision API**:
  - Claude Vision (claude-sonnet-4-6) — 패션 아이템 인식, 한국어 설명
  - GPT-4o Vision — 동일 영역
  - 비용: $0.003-0.01 / 분석
- **이미지 생성**:
  - OpenAI gpt-image-1 — 사진 입력 + 프롬프트로 편집 가능 (API 미확인, ChatGPT UI에서만 검증)
  - 비용: $0.04-0.08 / 장
- **카드 합성 백업**:
  - html2canvas (클라이언트) — DOM → PNG
  - satori (서버, Vercel @vercel/og 기반) — JSX → SVG → PNG, Edge 호환
  - Google Fonts: Caveat, Pacifico, **Nanum Pen Script** (한국어 손글씨)
- **저장**: Cloudinary 또는 Supabase Storage (선택: Supabase, DB 통합)

### Agent C — 사용자 페인포인트

- **기록 마찰**: 촬영 → 편집 → 태깅 → 업로드 = 10-15분 / 작심삼일 패턴
- **크리에이터 페인**: "이 옷 어디꺼?" 댓글/DM 폭주, 링크트리/LTK도 이탈률 높음
- **검색 페인**: 갤러리에서 과거 착장 검색 불가
- **MZ 트렌드**: 조용한 럭셔리, 빈티지 믹스 → 아이템 출처 = 콘텐츠 가치
