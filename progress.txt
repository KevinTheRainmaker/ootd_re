- OOTD-re: 착장 사진 1장 → AI Vision 분석 → 손글씨 카드 → /share/[id] 공유

## Codebase Patterns
- Next.js 14 App Router: src/app/ 구조, src/types/ 도메인 타입 분리
- Supabase 클라이언트 분리: supabase (anon, 클라이언트용) / supabaseAdmin (service_role, API Routes 전용)
- API 타입은 src/types/api.ts, 도메인 타입은 src/types/index.ts로 분리
- create-next-app이 기존 파일 충돌 시: 임시 디렉토리 생성 후 필요 파일만 복사
