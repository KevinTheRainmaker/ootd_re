# CPS (Context-Problem-Solution)

## Context

**누가**: 1인 개발자 사이드 프로젝트  
**플랫폼**: Flutter (iOS + Android 크로스플랫폼 동시 출시)  
**환경**: 패션에 관심 있는 MZ세대가 매일 코디를 기록하고 싶지만 수작업이 번거로운 시장. Instagram/Pinterest 등 비주얼 SNS에서 패션 콘텐츠 소비가 활발함.

## Problem

**핵심 문제**: 일상 코디를 기록하는 과정이 수작업으로 번거로워 지속적인 기록이 이루어지지 않음  
**구체적 페인포인트**:

- 사진 촬영 → 아이템 태깅 → SNS 업로드 → 날씨 메모까지 평균 10-15분 소요
- 옷장 전체 현황 파악이 어려워 중복 구매 발생
- 날씨에 맞는 코디 추천을 받을 AI 도구 부재
- 스타일 발견/공유와 개인 기록을 하나의 앱에서 할 수 없음

**정량적 지표**: 미정 (MVP 검증 후 도출 예정)

## Solution

**해결 후 기대 상태**: 사진 한 장 업로드로 AI가 아이템을 인식하고, 날씨 기반 코디를 추천하며, 커뮤니티에서 스타일을 발견할 수 있는 올인원 패션 플랫폼

**성공 지표**: 인터뷰에서 도출 예정  
**플랫폼**: Flutter (iOS + Android)  
**비즈니스**: Freemium (무료 기본 + Pro 구독)
